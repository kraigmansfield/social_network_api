# social_network_api

## Description

This is an API for a social network applicationh in which users can share their thoughts, react to friends, and create a friends list. This uses Express.js, MongoDB, Mongoose for its framework.

## Installation

```
npm init -y
npm i
npm express
npm mongoose
node server.js
```

## Technologies

Javascript
Express.js
Node.js
MongoDB
Mongoose
Insomnia
